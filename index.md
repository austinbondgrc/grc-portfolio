## 📌 What This Repository Is

Certifications prove theoretical knowledge. This portfolio proves execution capability.

This repository contains a fictional but highly realistic end-to-end Information 
Security Management System (ISMS) implementation for MunichTech GmbH — a 
hypothetical B2B Financial Data Analytics SaaS provider headquartered in Munich, 
Germany, operating under ISO 27001:2022, the EU NIS2 Directive, and DSGVO.

Every artifact has been built to a standard suitable for presentation to a 
DAkkS-accredited certification body, enterprise banking clients, and BaFin-regulated 
procurement teams.
